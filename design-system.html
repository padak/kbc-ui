Change Management Workflow

  • Step 1: Identify what you want to change (e.g., "make primary color darker")
  • Step 2: Find the corresponding design token in the <style> section
  • Step 3: Update the token value
  • Step 4: All components using that token update automatically

Example Changes

  • Change primary blue: Update --color-primary-500
  • Increase all spacing: Change --space-unit from 4px to 8px
  • Make text larger: Adjust --font-size-base
  • Rounder corners: Increase --radius-base
  • Different font: Change --font-family-base

How to Use Design Tokens in Components

In your Tailwind config, reference these tokens:

// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: {
          50: 'var(--color-primary-50)',
          500: 'var(--color-primary-500)',
          700: 'var(--color-primary-700)',
          // ... etc
        }
      },
      spacing: {
        1: 'var(--space-1)',
        2: 'var(--space-2)',
        // ... etc
      }
    }
  }
}

Then in components:

<button className="bg-primary-500 text-white px-4 py-2 rounded-md">
  Click me
</button>

How to Use Lucide React Icons

Always use Lucide React icons - never hard-code SVGs. Consistent design, tree-shakeable, 1000+ icons.

1. Import from lucide-react:

import { Database, Activity, Sparkles, CheckCircle2 } from 'lucide-react'

2. Use in components with size classes:

{/* Small icons: 16px (w-4 h-4) */}
<Sparkles className="w-4 h-4" />

{/* Medium icons: 20px (w-5 h-5) */}
<Activity className="w-5 h-5" />

{/* Large icons: 24px (w-6 h-6) */}
<Database className="w-6 h-6" />

3. Apply colors with Tailwind or CSS variables:

{/* Using Tailwind classes */}
<CheckCircle2 className="w-4 h-4 text-success-500" />

{/* Using CSS variables for AI theme */}
<Sparkles
  className="w-5 h-5"
  style={{ color: 'var(--color-ai-purple)' }}
/>

4. Common icons for this project:

  • Database, Table (Storage)
  • Activity, Clock (Jobs)
  • Workflow, GitBranch (Flows)
  • Users, UserCog (Team)
  • Sparkles, Zap (AI features)
  • CheckCircle2, AlertCircle, X (Status/Actions)
